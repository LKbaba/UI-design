### 字体

| 用途 | 字体族 |
|------|--------|
| 标题 | system-ui, -apple-system, sans-serif |

| 数据/代码 | 'SF Mono', 'Cascadia Code', 'Consolas', monospace |

### 风格偏好

- **间距**：紧凑（仪表盘信息密度高）
- **圆角**：8px（面板）/ 4px（小元素）
- **阴影**：无（用边框分隔，深色主题下阴影无意义）
- **暗色模式**：默认就是暗色
- **动效**：适度（数据流动感，尊重 prefers-reduced-motion）

---

## 硬约束

- 纯 HTML + CSS + JS，无外部依赖（不引入 CDN 库）
- 全屏 100vh，零滚动
- 不编造真实品牌数据，使用占位模拟数据
- 图标使用内联 SVG（Lucide 风格，统一线宽 1.5px）

## 验收标准

- [ ] 全屏 100vh，无滚动条
- [ ] CSS Grid 布局，面板可视区域清晰
- [ ] 响应式：1440px / 1024px / 768px 三个宽度无异常
- [ ] 对比度：#FAFAFA 对 #0A0A0A ≥ 15:1
- [ ] 模拟数据用 JS 动态生成，不编造真实品牌
- [ ] 无浏览器控制台报错
- [ ] 动效尊重 prefers-reduced-motion
- [ ] 图标统一为内联 SVG
